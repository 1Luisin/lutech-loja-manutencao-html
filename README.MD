# 🖥️ Lutech - Loja de Manutenção

Projeto de site fictício de uma loja de manutenção e reparo de computadores, criado com HTML.

## Páginas
- Página inicial
- Serviços oferecidos
- Contato

## Tecnologias
- HTML5

## Objetivo
Praticar estruturação de páginas web e organização semântica usando apenas HTML.
